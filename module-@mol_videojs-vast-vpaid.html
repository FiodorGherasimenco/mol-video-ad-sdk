    vastVpaidPlugin(options)
    videojs-vast-vpaid/src/vastVpaidPlugin.js, line 61

    VideoJS Vast Vpaid plugin class

    Parameters:
    Name Type Description
    options Object

    Options Map. The allowed properties are:

    Properties
    Name Type Attributes Description
    getAdTag function

    will be called to get the vast tag must return a vast tag (string) or a promise that will resolve with a vast tag.
    placeholder HTMLElement <optional>

    placeholder element that will contain the video ad. If not provider the plugin will create one.
    adStartEvent string <optional>

    name of the event to run an ad. The plugin try to run an ad if that event is triggered by the player. Defaults to adStart
    adStartedEvent string <optional>

    name of the event the player will trigger to indicate an ad has started. Defaults to adStarted

    Properties
    Name Type Attributes Description
    adUnit string <optional>

    the video ad unit will be added to the adStartedEvent event object.
    adCancelEvent string <optional>

    name of the event to cancel an ad. The plugin will cancel any running ad if that event is triggered by the player Defaults to adCancel
    adRunFinishEvent string <optional>

    name of the event the player must trigger to indicate that the ad finished. Will be called if an ad completes or gets cancel or has an error. Defaults to adRunFinish
    adErrorEvent string <optional>

    name of the event the player must trigger to indicate that there was an error playing the video ad. Defaults to adError
    onAdStart runWaterfall~onAdStart <optional>

    will be called once the ad starts with the ad unit.
    onRunFinish runWaterfall~onRunFinish <optional>

    will be called whenever the ad run finishes. Can be used to know when to unmount the component
    logger Console <optional>

    Optional logger instance. Must comply to the Console interface. Defaults to window.console
    wrapperLimit number <optional>

    Sets the maximum number of wrappers allowed in the VastChain. Defaults to 5.
    viewability boolean <optional>

    if true it will pause the ad whenever is not visible for the viewer. Defaults to false
    responsive boolean <optional>

    if true it will resize the ad unit whenever the ad container changes sizes. Defaults to false
    timeout number <optional>

    timeout number in milliseconds. If set, the video ad will time out if it doesn't start within the specified time.
    tracker TrackerFn <optional>

    If provided it will be used to track the VAST events instead of the default pixelTracker.
    hooks Object <optional>

    Optional map with hooks to configure the behaviour of the ad.

    Properties
    Name Type Attributes Description
    createSkipControl function <optional>

    If provided it will be called to generate the skip control. Must return a clickable HTMLElement that is detached from the DOM.
