    @mol/react-vast-vpaidVideoAd(props)

    new VideoAd(props)
    react-vast-vpaid/src/VideoAd.js, line 27

    React component to lad a vast/vpaid video ad.

    Parameters:
    Name Type Description
    props Object

    react props.

    Properties
    Name Type Attributes Description
    getTag function

    must return an ad tag or a promise that will resolve with an ad tag.
    children React.nodes <optional>

    whatever you pass as children will be displayed while the ad is loading.
    skipControl function | HTMLElement <optional>

    If a function is provided it will be called to generate the skip control. The skip control must be a clickable HTMLElement that is detached from the DOM.
    height number <optional>

    the component will resize the ad unit if the passed height changes.
    width number <optional>

    the component will resize the ad unit if the passed width changes.
    logger Console <optional>

    must comply to the Console interface. Defaults to window.console
    onAdStart runWaterfall~onAdStart <optional>

    will be called once the ad starts with the ad unit.
    onError runWaterfall~onError <optional>

    will be called if there is an error with the video ad with the error instance.
    onRunFinish runWaterfall~onRunFinish <optional>

    will be called whenever the ad run finishes. Can be used to know when to unmount the component
    tracker TrackerFn <optional>

    If provided it will be used to track the VAST events instead of the default pixelTracker.
    videoElement HTMLVideoElement <optional>

    If provided it will be used to display the video ad. Beware that it will not clean the sources after the ad is finished.
    viewability boolean <optional>

    if true it will pause the ad whenever is not visible for the viewer. Defaults to false
    responsive boolean <optional>

    if true it will resize the ad unit whenever the ad container changes sizes. Defaults to false
